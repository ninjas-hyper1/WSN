13. Creating Public Database Links at the Central Machine:

CREATE PUBLIC DATABASE LINK sales_link1
CONNECT TO sales_user IDENTIFIED BY ckt
USING 'XE';

CREATE PUBLIC DATABASE LINK sales_link2
CONNECT TO sales_dtls IDENTIFIED BY ckt
USING 'XE';

14. Retrieving all the records from sales_link1 and sales_link2:

SELECT
    sales_order.order_id@sales_link1,
    sales_order.client_name,
    sales_order.order_date,
    sales_order_details.product_name,
    sales_order_details.quantity,
    sales_order_details.price
FROM
    sales_order@sales_link1,
    sales_order_details@sales_link2
WHERE
    sales_order.order_id@sales_link1 = sales_order_details.order_id@sales_link2;
